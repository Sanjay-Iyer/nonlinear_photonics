# nextnano++ run log

Append one row per meaningful run (smoke tests, sweep points worth
remembering, anything you had to debug). This is the hand-kept ledger that
ties a git commit to what actually executed on which machine, since `output/`
itself is never committed.

Get the short SHA with `git rev-parse --short HEAD`.
